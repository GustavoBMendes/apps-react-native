Conjunto de aplicações desenvolvidas durante o curso de desenvolvimento multiplataforma Android/IOS com React e Redux do professor Jorge Sant Ana.

Para iniciar algum projeto:
> react-native init 'nome do projeto'
> cd 'nome do projeto'
> react-native start
abrir simulador
code!

Instalação de dependências para os projetos:

React navigation:
> npm install @react-navigation/native
> npm install react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view
> npm install @react-navigation/stack

Firebase:
> npm install --save firebase

Redux:
> npm install --save redux
> npm install --save react-redux

Redux Thunk:
> npm install --save redux-thunk

Base64:
> npm install --save base-64

Tab View:
> npm install --save react-native-tab-view
